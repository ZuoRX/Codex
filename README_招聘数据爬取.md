# 中国招聘网站数据分析岗位爬取系统

## 项目概述

本系统用于自动化爬取中国主要招聘平台上**数据分析相关职位**的详细信息（2025-2026年），
目标获取 **10,000+ 条**职位数据，清洗整理后输出为格式化Excel报告。

### 支持的招聘平台

| 平台 | 网址 | 特点 |
|------|------|------|
| BOSS直聘 | zhipin.com | 互联网/科技岗位丰富 |
| 智联招聘 | zhaopin.com | 覆盖面广，数据量大 |
| 前程无忧 | 51job.com | 传统企业职位多 |
| 猎聘网 | liepin.com | 中高端职位为主 |
| 拉勾网 | lagou.com | 互联网技术岗专业平台 |

### 爬取的职位关键词

- 数据分析、数据分析师
- 数据运营、商业分析
- BI分析、业务分析
- 数据挖掘

### 覆盖城市

北京、上海、深圳、广州、杭州、成都、武汉、南京、西安、苏州、重庆、天津等15+主要城市

## 数据字段说明

| 字段 | 说明 |
|------|------|
| 职位名称 | 招聘职位名称 |
| 薪资范围 | 原始薪资字符串（如：15-25K·13薪） |

| 薪资上限_千元 | 解析后薪资上限（单位：千元/月） |
| 薪资中位数_千元 | （下限+上限）/ 2 |
| 薪资档次 | 薪资区间分类（8K以下/8-12K/12-18K/18-25K/25-35K/35K以上） |
| 工作城市 | 工作所在城市 |
| 工作区域 | 城市内具体区域 |

| 公司规模 | 公司员工规模 |
| 公司类型 | 公司性质（国企/民营/外资等） |
| 行业类别 | 所属行业 |
| 经验要求 | 工作年限要求 |
| 学历要求 | 最低学历要求 |

| 技能要求 | 提取的技能关键词（Python/SQL/Tableau等） |
| 福利待遇 | 职位福利标签 |
| 招聘人数 | 招聘人数 |

| 职位链接 | 职位详情页URL |
| 爬取时间 | 数据爬取时间戳 |

## Excel输出结构

生成的Excel包含以下6个工作表：

1. **招聘数据_全量** - 所有职位数据（带格式）
2. **薪资分析** - 薪资统计、各城市薪资对比、经验与薪资关系
3. **城市分布** - 各城市职位数量、学历要求分布
4. **技能需求分析** - 技能频次统计、不同学历对应技能需求
5. **公司分析** - 公司规模/行业分布、招聘最多公司Top30
6. **数据来源统计** - 各平台数据量、数据完整度

## 安装与使用

### 环境要求

- Python 3.8+
- 需要访问目标网站的网络环境（国内网络）
- Chromium/Chrome 浏览器

### 安装依赖

```bash
pip install -r requirements.txt

# 安装Playwright浏览器
python -m playwright install chromium
```

### 运行爬虫

```bash
# 爬取所有网站（推荐）
python main.py

# 只爬取指定网站
python main.py --sites boss zhaopin

# 测试模式（少量数据，验证功能）
python main.py --test

# 只处理已有原始数据（跳过爬取）
python main.py --process-only

# 可见浏览器模式（调试用）
python main.py --headless False --sites boss

# 指定输出路径
python main.py --output /path/to/output.xlsx
```

## 反爬机制说明

本系统采用以下方式应对各招聘平台的反爬检测：

1. **浏览器指纹伪装**
   - 隐藏 `navigator.webdriver` 属性
   - 伪造 plugins、languages、platform 等浏览器属性
   - 模拟真实的 WebGL 指纹
   - 随机 User-Agent 轮换

2. **行为模拟**
   - 随机页面滚动（模拟用户阅读行为）
   - 页面间随机延迟（2-5秒）
   - 人工化鼠标移动

3. **API拦截**
   - 对智联招聘、前程无忧等通过拦截API响应直接获取数据
   - 避免对DOM解析的依赖，提高稳定性

4. **自动重试**
   - 检测反爬页面（验证码、访问限制等）
   - 触发反爬时等待30秒后重试
   - 最大重试3次

## 注意事项

1. **请合理控制爬取频率**，避免对目标网站造成过大压力
2. 部分网站（如BOSS直聘）需要登录才能查看完整数据，建议手动登录后使用Cookie
3. 如遇验证码，可将 `--headless False` 改为可见模式手动处理
4. 原始数据自动保存到 `data/raw/` 目录，爬取中断后可从断点继续
5. 最终Excel文件保存在 `output/` 目录

## 数据处理流程

```
爬取 → 原始JSON → 数据清洗 → 去重 → 字段标准化 → 统计分析 → Excel导出
                                ↓
                    - 薪资解析（统一为千元/月）
                    - 城市名称标准化
                    - 经验/学历要求归一化
                    - 技能关键词提取
                    - 薪资档次分类
```
